## Redux의 3요소

- Provider: 맨처음 프로젝트에 redux 를 적용시켜준다. store를 props으로 받아야한다.
- useSelector: 함수를 인자로 받으며, redux store 의 상태를 사용할 수 있게 해준다. 무선연결!

```js
const number = useSelector((state) => state.number);
```

- useDispatch: dispatch 를 불러와 액션을 호출할 수 있게 한다.

```js
const dispatch = useDispatch();
onClick={() => { dispatch({ type: 'PLUS' }) }} // type 은 대문자 const로 관리하자
```
